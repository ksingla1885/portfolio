# 🚀 Aryan Mittal — Full-Stack Portfolio

A high-performance, minimalist, and engineering-focused portfolio designed for the modern web. Built with the **MERN Stack** and **Vite**, this project showcases a personal brand with a "hacker/terminal" aesthetic and custom-architected UI components.

## ✨ Key Features

- **Custom Brand Identity**: Bespoke "Technical Core" favicon and neon yellow/dark color palette.
- **Micro-Animations**: Custom cursor interactions and scroll-reveal system for an immersive experience.
- **Engineering Mindset**: Sections highlighting system thinking, scalability, and code craft.
- **Beyond Code**: A dashboard-like lifestyle section reflecting personal interests and technical curiosity.
- **Full Responsiveness**: Seamlessly transitions across ultra-wide monitors, tablets, and mobile devices.
- **Modern Typography**: Curated font stack featuring *Syne*, *Bebas Neue*, and *JetBrains Mono*.

## 🛠️ Tech Stack

- **Core**: React.js, Vite
- **Styling**: Vanilla CSS (Custom tokens, Glassmorphism)
- **State Management**: React Hooks
- **Animations**: CSS Transitions/Keyframes + Scroll Reveal Intersection Observer
- **Deployment**: Ready for Netlify / Vercel

## 🚀 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/heyaryanmittal/Portfolio.git
    cd Portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Locally**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🏗️ Project Structure

- `src/App.jsx`: Main application architecture and content layer.
- `src/index.css`: Global design system, animation tokens, and media queries.
- `public/favicon.svg`: Custom-designed technical brand icon.
- `public/assets/`: Optimized project and personal assets.
